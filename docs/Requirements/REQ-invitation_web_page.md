# 要件書：招待機能 中間Webページ実装（Next.js）

## 概要

招待リンクをLINEで共有したとき、ブラウザで開く中間Webページを作成する。
既存の Next.js プロジェクト（Vercel）に `/invite/[token]` ページを追加する。

## ユーザーストーリー

- 招待されたユーザーとして、LINEで受け取ったリンクをブラウザで開いたとき、イベント情報と参加ボタンが表示されてほしい
- アプリがインストール済みの場合はアプリを起動して参加、未インストールの場合はApp Storeへ誘導してほしい

## スコープ

### URLパターン

```
https://[ドメイン]/invite/[token]
```

### 表示内容

APIから取得して表示する情報：
- 招待者名：「〇〇さんがイベントに招待しています」
- イベント名
- 招待コード（ABC-1234形式、大きく表示）

### ボタン・アクション

**「アプリで参加する」ボタン**
- カスタムURLスキーム `michimark://invite/[token]` を起動
- 2秒後にアプリが反応しない場合、App Storeへ誘導するフォールバック処理を実装

**「App Storeからダウンロード」リンク**
- 常時表示

### エラー状態の表示

| エラー種別 | 表示メッセージ |
|---|---|
| `expired` | 「この招待リンクの有効期限が切れています」 |
| `used_up` | 「この招待リンクは使用済みです」 |
| `not_found` | 「招待リンクが見つかりません」 |

### OGPメタタグ

LINEでリンクをシェアしたとき綺麗に表示されるよう設定する。

| タグ | 値 |
|---|---|
| `og:title` | 「[イベント名]への招待」 |
| `og:description` | 「[招待者名]さんがミチマークのイベントに招待しています」 |

### 技術仕様

- `getServerSideProps` でAPIからイベント情報を取得
- Tailwind CSS でスタイリング
- ミチマークのブランドカラーに合わせたデザイン

## 前提・制約

- 既存の Next.js プロジェクト構成を確認してから実装すること
- INV-1（バックエンド）の完了が前提
- AppStore 無料版リリース（REL-1）完了後に着手する
