# 要件書：招待機能 招待コード入力画面（Flutter）

## 概要

招待コード（XXX-9999形式）を手入力してイベントに参加できる画面を Flutter で実装する。

## ユーザーストーリー

- 招待されたユーザーとして、招待コードを手入力してイベントに参加したい
- コードが無効・期限切れ・使用済みの場合は適切なエラーメッセージを確認したい

## スコープ

### 画面：InviteCodeInputView

**入力フィールド**
- 招待コード入力（XXX-9999形式）
- 自動大文字変換

**ボタン**
- 「参加する」ボタン
- ローディング状態の表示

**表示エリア**
- エラーメッセージ表示エリア

### バリデーション

- 入力形式チェック：英字3文字-数字4桁（正規表現）
- API コール前にフォーマットチェックを行う

### API呼び出し

```
POST /invitations/code
Body: { code: "ABC-1234", user_id: [ログイン中のユーザーID] }
```

### 成功時の動作

- 「[イベント名]に参加しました！」アラート表示
- イベント詳細画面へ遷移（`event_id` を使って）

### エラー時の動作

| エラー種別 | 表示メッセージ |
|---|---|
| `expired` | 「この招待コードの有効期限が切れています」 |
| `used_up` | 「この招待コードは使用済みです」 |
| `not_found` | 「招待コードが見つかりません」 |
| `already_joined` | 「すでにこのイベントに参加しています」 |

### 画面への導線

- 既存のイベント一覧画面またはメニューに「招待コードで参加」ボタンを追加
- 追加箇所は既存のUI構成を確認して自然な位置を選ぶ

## 前提・制約

- 設計憲章に従い BLoC パターンで実装する
- AppStore 無料版リリース（REL-1）完了後に着手する
