# 要件書: MichiInfo Mark/Link削除UI変更

- **要件ID**: REQ-michi_info_delete_icon
- **作成日**: 2026-04-12
- **担当**: product-manager
- **ステータス**: 確定
- **種別**: UI改善（UI-3）

---

## 1. 背景・目的

MichiInfo画面のMarkカード・Linkカードのスワイプ削除UIを廃止し、削除アイコンを常時表示する方式に変更する。

スワイプへの気づきにくさを解消し、削除操作を直感的にする。

---

## 2. 要件一覧

### 2.1 廃止する操作

- MichiInfo画面のMarkカード・Linkカードの `flutter_slidable` スワイプ削除UI

### 2.2 削除アイコン

| 項目 | 内容 |
|---|---|
| アイコン | ゴミ箱（Icons.delete 等） |
| 背景色 | 赤（`#FEE2E2` 相当） |
| アイコン色 | 赤（`#DC2626` 相当） |
| 配置 | カード右端に常時表示 |
| 確認ダイアログ | なし（タップ即削除） |

### 2.3 給油アイコンの配置変更（タイムライン接点への統合）

現在カード行内に表示している給油アイコン（⛽）を、MichiInfo タイムラインの**縦罫線接点ドット**に統合する。

| 項目 | 内容 |
|---|---|
| 給油アイコン表示位置 | 縦罫線の区間接点ドット（タイムライン上のノード）の内部 |
| 接点ドットのサイズ | 給油アイコンを内包できるよう拡大（現行の点より大きく） |
| 接点ドットの縦幅 | ActionTime表示の縦幅分を加算して大きくする |
| 給油なし時 | 接点ドットは現行サイズを維持 |

---

## 3. 影響範囲

| 対象 | 変更内容 |
|---|---|
| MarkCard・LinkCard | スワイプUI撤去・右端に削除アイコン（案A）常時表示 |
| タイムライン縦罫線（CustomPainter/Canvas） | 区間接点ドットのサイズ変更・給油ありの場合は内部に給油アイコンを表示 |
| カード内の給油アイコン表示 | 廃止（接点ドットに移動） |

---

## 4. 対象外

- 削除後のデータ処理（既存実装に準ずる：REQ-michi_info_card_delete.md 参照）
- 削除取り消し（Undo）
- ActionTime自体の表示仕様変更
