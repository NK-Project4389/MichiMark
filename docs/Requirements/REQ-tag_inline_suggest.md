# 要件書: タグインラインサジェスト入力

**要件ID**: REQ-tag_inline_suggest
**作成日**: 2026-04-07
**ステータス**: 実装済み

---

## 背景・目的

現状、イベント詳細の基本画面でタグを選択するには選択画面（別画面）への遷移が必要で、操作ステップが多い。
タグ入力をインライン（同一画面内）で完結できるようにし、UXを改善する。
また、タグマスタに存在しない新しいタグも入力時に即座に作成・紐づけできるようにする。

---

## ユーザーストーリー

- As a ユーザー, I want to 基本画面のタグフィールドに直接テキスト入力してタグを追加したい, so that 選択画面に遷移せずにタグ設定を完了できる
- As a ユーザー, I want to 入力中に既存タグのサジェストを表示してほしい, so that 既存タグを素早く選択できる
- As a ユーザー, I want to マスタにないタグを入力して確定すると自動追加してほしい, so that 新しいタグを素早く追加できる

---

## 機能要件

### FR-001: インライン入力フィールド
- タグ項目を `_SelectionRow`（選択画面遷移型）からテキスト入力フィールドに変更する
- 入力フィールドは基本画面内にインラインで表示する
- 画面遷移は発生させない

### FR-002: 選択済みタグのチップ表示
- 選択済みタグは `Chip` ウィジェットで表示する
- 各チップに削除ボタン（×）を設け、タップで選択解除できる

### FR-003: リアルタイムサジェスト
- テキスト入力変化を監視し、タグマスタを部分一致で検索してサジェストリストを表示する
- 既に選択済みのタグはサジェストから除外する
- 入力が空の場合はサジェストを非表示にする

### FR-004: サジェストからの選択
- サジェストリストのアイテムをタップするとタグが選択状態になる
- 選択後、入力フィールドをクリアしてサジェストを非表示にする

### FR-005: キーボード確定による追加
- テキストフィールドでキーボードの確定（submitAction）を押すとタグを追加する
- 既存マスタに完全一致するタグがある場合はそれを使用する
- マスタに存在しない値の場合はタグマスタに新規追加してからイベントに紐づける

### FR-006: 重複チェック
- 既に選択済みのタグと同名のタグを追加しようとした場合は無視する
- タグマスタへの新規追加時も同名の既存タグがないか確認する

### FR-007: タグ選択画面の保持
- 既存のタグ選択画面（`/selection` + `SelectionType.eventTags`）は削除しない
- 基本画面からのタグ選択画面への参照（`BasicInfoOpenTagsSelectionDelegate`）を切るだけにする

---

## 非機能要件

- タグマスタは画面読み込み時に全件キャッシュし、入力変化のたびにDBを叩かない
- 部分一致検索は大文字・小文字を区別しない

## 影響範囲

| ファイル | 変更種別 |
|---|---|
| `basic_info_event.dart` | 新イベント追加（TagInputChanged, SuggestionSelected, InputConfirmed, TagRemoved） |
| `basic_info_state.dart` | `BasicInfoLoaded` に `allTags`, `tagSuggestions` フィールド追加 |
| `basic_info_bloc.dart` | `TagRepository` DI追加・新ハンドラ実装 |
| `basic_info_view.dart` | タグ行を `_TagInputSection` に置き換え |
| `event_detail_page.dart` | `BasicInfoBloc` 生成に `tagRepository` を追加 |

---

## 対応しないもの

- タグ選択画面の削除・変更
- 他画面（マーク詳細・リンク詳細等）のタグ入力変更
