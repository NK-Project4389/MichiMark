# 要件書: BasicInfo 参照/編集モード切替UI

- **要件ID**: REQ-basic_info_tap_to_edit
- **作成日**: 2026-04-12
- **担当**: product-manager
- **ステータス**: 確定
- **種別**: UI改善（UI-2）

---

## 1. 背景・目的

BasicInfoセクションの編集アイコンを廃止し、セクション全体タップで参照/編集モードを切り替えるUIに変更する。

UIをすっきりさせつつ、タップで編集できることをユーザーが直感的に把握できるようにする。

---

## 2. 要件一覧

### 2.1 廃止

- BasicInfoセクションの編集アイコン（`Icons.edit` 等の `IconButton`）

### 2.2 参照モード（デフォルト）

| 項目 | 内容 |
|---|---|
| 表示 | BasicInfoセクション全体を読み取り専用で表示 |
| 背景 | 薄いTeal背景（`#E8F4F8` 相当）でセクション全体を囲む |
| ヒントテキスト | セクション下部に「タップして編集」テキストを表示 |
| タップ領域 | セクション全体（どこをタップしても編集モードに切り替わる） |

### 2.3 編集モード

| 項目 | 内容 |
|---|---|
| 切り替えトリガー | 参照モードのセクション全体タップ |
| 表示 | 全フィールドが編集可能な入力UI（既存の `_BasicInfoForm`）に切り替わる |
| 保存ボタン | フォーム下部にインライン表示（右寄せ or 中央寄せ） |
| キャンセルボタン | 保存ボタンの左に並べてインライン表示 |
| キャンセル動作 | 変更を破棄して参照モードに戻る |
| 保存動作 | DB保存後に参照モードに戻る |

---

## 3. 実装メモ（architect向け）

- 既存の `BasicInfoDraft.isEditing` フラグと `_BasicInfoReadView` / `_BasicInfoForm` の切り替え構造をそのまま活かす
- 変更箇所は参照モードのスタイル調整と `IconButton(Icons.edit)` の削除のみ

---

## 4. 影響範囲

| 対象 | 変更内容 |
|---|---|
| BasicInfoセクション（参照モード） | Teal薄背景・「タップして編集」ヒント追加・セクション全体タップ可能に |
| BasicInfoセクション（編集モード） | 保存・キャンセルボタンをフォーム下部インラインに移動 |
| 編集アイコン（`IconButton`） | 廃止 |

---

## 5. 対象外

- BasicInfo以外のセクションのモード切替
- 編集画面の入力フィールド内容の変更
