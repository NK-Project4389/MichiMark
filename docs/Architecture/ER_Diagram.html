🗄 Firestore コレクション構造
users /{uid}
├─ profile （1件固定）
uid, isAppleLinked, schemaVersion
 
├─ members /{memberId}
memberName, mailAddress, isVisible, isDeleted
 
├─ trans /{transId}
transName, kmPerGas, meterValue, isVisible, isDeleted
 
├─ tags /{tagId}
tagName, isVisible, isDeleted
 
├─ actions /{actionId}
actionName, fromState, toState
isToggle, togglePairId, needsTransition
 
├─ topics /{topicId}
topicName, topicType, color, isVisible, isDeleted
 
└─ events /{eventId}
eventName, transId, kmPerGas, pricePerGas
payMemberId, topicId
memberIds: [ ] <-- event_members をArray化
tagIds: [ ] <-- event_tags をArray化
isDeleted
 
├─ markLinks /{markLinkId}
markLinkSeq, markLinkType, markLinkDate
markLinkName, meterValue, distanceValue, memo
memberIds: [ ] <-- mark_link_members をArray化
actionIds: [ ] <-- mark_link_actions をArray化
isFuel, pricePerGas, gasQuantity, gasPrice, gasPayerId
isDeleted
 
├─ payments /{paymentId}
paymentSeq, paymentAmount, paymentMemberId, paymentMemo
splitMemberIds: [ ] <-- payment_split_members をArray化
isDeleted
 
└─ actionTimeLogs /{logId}
actionId, timestamp, isDeleted
※ イベント全体のログ。MarkLink単位ではなくEvent単位で管理。
　アクション現在状態 = 全logのtimestamp順集計で決定。

🛡 Security Rules（概要）
match /users/{uid}/{document=**} {
allow read, write: if
request.auth != null
&& request.auth.uid == uid;
}
