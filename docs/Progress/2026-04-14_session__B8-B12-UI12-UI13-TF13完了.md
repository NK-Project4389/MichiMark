## 技術メモ

### B-10 修正ポイント
- `EventDomain.members` はイベント保存時点のスナップショット → `isVisible` が連動しない
- 対策: MemberRepository を直接 inject して最新マスタと突合フィルタ
- 修正ファイル: michi_info_bloc / payment_detail_bloc / event_detail_page / router

### UI-12 実装ポイント
- EventDetailBloc に `isNewEvent: bool` と `isSavedAtLeastOnce: bool` を追加
- 子Bloc（BasicInfo/MichiInfo/PaymentInfo）の SavedDelegate → `EventDetailChildSaved` event で通知
- 戻るボタン押下時に未保存判定 → `EventRepository.deleteEvent` を呼んでサイレント削除

### UI-13 実装ポイント
- 各DetailBloc State に `initialDraft`（起動時スナップショット）と `showCancelConfirmDialog: bool` を追加
- `DismissPressed` で `currentDraft != initialDraft` を比較 → 差分ありなら CupertinoAlertDialog 表示
- Stack によるオーバーレイ方式（showDialog ではなく BlocBuilder の builder 内）
