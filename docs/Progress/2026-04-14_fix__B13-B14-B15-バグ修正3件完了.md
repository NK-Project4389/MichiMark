## 修正内容詳細

### B-13: BasicInfoマスタ非表示フィルタ漏れ
- **原因**: B-10はMarkDetail/LinkDetail/PaymentDetailのみ修正。BasicInfoBlocにisVisibleフィルタが未実装だった
- **修正**: `basic_info_bloc.dart` の `_onStarted` でallMembers/allTags/allTransに `isVisible` フィルタ追加
- **追加修正**: `_buildInitialMemberSuggestions` のスナップショットをvisibleIdsセットでフィルタ
- **seed_data修正**: `trans-002` に `meterValue: 0` を追加（Transバリデーション通過）

### B-14: プラスボタン再押し不可
- **原因**: TopicSelectionSheet をキャンセルしても `showTopicSelection: true` がリセットされない → 次回プラス押下でEquatableにより同一State判定 → BlocがEmitしない → シートが開かない
- **修正**: `EventListTopicSelectionDismissed` イベントを追加、キャンセル時にdispatchして `showTopicSelection: false` にリセット
- **修正ファイル**: `event_list_event.dart`, `event_list_bloc.dart`, `event_list_page.dart`

### B-15: 燃費推定 経費合計に反映されない
- **原因**: `EventDetailOverviewAdapter.toMovingCostProjection()` の `totalPaymentLabel` が `result.totalPayment`（給油実績）のみ参照。燃費推定モードでは給油実績がnull → `---` 表示
- **修正**: `effectiveGasPrice`（推定値）を `result.totalPayment` のフォールバックとして使用
- **修正ファイル**: `event_detail_overview_adapter.dart`
