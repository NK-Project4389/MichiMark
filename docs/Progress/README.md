# 進捗記録ディレクトリ

## 目的

- 作業ログをセッションごとに記録する
- 次回の会話開始時に読み込むことでコンテキスト消費を抑える

## 運用ルール

- ファイル名: `YYYY-MM-DD_[作業内容].md`
- 新しいセッション開始時に最新のログを参照してから作業開始する
- 完了した作業・未完了の作業・次回やること を必ず記載する
