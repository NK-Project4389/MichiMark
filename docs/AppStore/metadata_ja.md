## 説明文（T-260e）

> 4000文字以内。

```
移動の記録・集計・管理をワンアプリで完結。

MichiMarkは、ドライブ・ツーリング・旅行・仕事の移動を記録して集計できるログアプリです。地点と区間を組み合わせてルートを記録し、タイムライン形式で振り返ることができます。

【こんな方におすすめ】

・ツーリング・ドライブの記録を残したい
・旅行の行程と費用をまとめて管理したい
・訪問介護・営業など、移動を仕事で使う
・暮らしのマーケット出店者として交通費を管理したい

【主な機能】

■ 移動ログ記録
地点と区間を登録してルートを記録。日時・メモ・タグを自由に設定できます。タイムライン形式のMichiInfoビューで移動の流れが一目でわかります。

■ トピック別管理
「ドライブ」「旅行」「移動コスト」など、用途に合わせたトピックでイベントを整理。トピックごとに表示される情報が最適化されます。

■ 旅費・支払い管理
支払い記録をメンバーごとに登録し、割り勘の精算金額を自動計算。旅行やツーリング仲間との費用管理がかんたんです。

■ 燃費・移動コスト集計
走行距離・燃費・ガソリン代を記録すると、移動コストを自動で集計。推計移動コストで旅の実費を把握できます。

■ メンバー・タグ管理
よく使うメンバーやタグを登録しておけば、記録のたびにすばやく入力。入力候補がリアルタイムで表示されます。

■ カスタム数値キーパッド
距離・金額の入力に特化したキーパッドを採用。四則演算もその場で計算できます。

【使い方のイメージ】

ツーリング仲間と出かけた日 —
出発地・立ち寄りスポット・目的地をマークで記録。区間ごとの走行距離とガソリン代を入力すれば、燃費と移動コストを自動集計。食事代や宿泊費もメンバーごとに登録して、帰りの精算もスムーズ。

仕事の移動記録に —
訪問先をマークで記録し、移動時間と交通費をタグで整理。月末の経費精算がグッとかんたんになります。

【プライバシー】

MichiMarkはすべてのデータをお使いのデバイス内にのみ保存します。個人情報の外部送信・収集は行いません。

プライバシーポリシー: https://michimark-web.vercel.app/privacy.html
サポート: https://michimark-web.vercel.app/support.html
```

## 備考・見直しポイント

- **サブタイトル**：30文字以内ルールあり。現在14文字でOK
- **キーワード**：重複NGワード注意（アプリ名・カテゴリ名と重複するものは効果なし）。「ナビゲーション」「旅行」はカテゴリ名と重複するため除外済み

- **プロモーションテキスト**：フィーチャーセールなどに合わせて随時更新可能（説明文と違い即時反映）
