## 説明文

> 4000文字以内。審査提出時に更新。

```
「あの日どこ行って、いくらかかったっけ？」
「旅費の精算、まだLINEで計算してる？」

MichiMarkは、移動の記録・費用管理・仲間との共有を
ひとつにまとめたログアプリです。

【旅行の記録も精算も、これひとつ】
地点・移動手段・費用をタイムラインで記録。
メンバーごとの旅費精算も自動で計算。
招待リンクひとつで仲間と共有できます。

【ダッシュボードで一目把握】
走行距離・コストをグラフで自動集計。
「今週どれだけ走った？いくら使った？」が
開くだけでわかります。

【営業・外回りの交通費管理に】
日々の移動をタイムラインで記録するだけ。
ダッシュボードで週・月のコストを自動集計。
経費精算の「あれいくらだっけ」がなくなります。

【訪問作業の売上・時間管理に】
到着→作業開始→終了→出発をワンタップ記録。
作業時間・売上・時給換算まで自動集計。
個人事業主・フリーランスの日々の現場管理に。

【基本機能】
・タイムライン形式で移動ログ記録
・タグ・メンバー・交通手段のマスター管理
・旅費の自動精算・収支バランス表示
・カレンダー表示で月別の旅行を振り返り
・距離・金額の入力に特化したカスタムキーパッド

【プライバシー】
MichiMarkはすべてのデータをお使いのデバイス内にのみ保存します。個人情報の外部送信・収集は行いません。

プライバシーポリシー: https://michimark-web.vercel.app/privacy.html
サポート: https://michimark-web.vercel.app/support.html
```
