現在のトピックへの推奨割り当て
2つの既存トピックタイプに対し、最適なパレットカラーを割り当てます。

movingCost（通勤・日常移動）

No.5 エメラルドグリーン を推奨
Primary #2E9E6B
Dark #1A7A52
Tint #D4F0E4
Flutter Color Color(0xFF2E9E6B)
ロゴのティールブルー（H=200°）から色相-45°に位置するエメラルドグリーン（H=155°）。 同系色の展開でアプリ全体の統一感を保ちながら、緑の「継続・安定・日常」のニュアンスが 通勤・通学など毎日の移動費という用途と一致する。ロゴとの色相差が45°で近すぎず遠すぎないバランス。

travelExpense（出張・旅行）

No.2 アンバーオレンジ を推奨
Primary #E07B39
Dark #B85C20
Tint #FDEBD4
Flutter Color Color(0xFFE07B39)
ロゴのティールブルーの分割補色（H=200° + 190° = H=30°）に当たるアンバーオレンジ。 寒色のティールと暖色のオレンジは高いコントラストを持ち、movingCostとの識別も明確。 オレンジの「活動・旅情・エネルギー」が出張・旅行という非日常感と直感的に一致する。

トピックの性質 推奨カラーNo 色名 理由
日常系（通勤・通学） No.5 エメラルドグリーン 継続・安定・日常の緑。毎日の移動費に自然なイメージ。
旅行系（旅行・出張） No.2 アンバーオレンジ 活動・旅情・非日常感。travelExpenseの直感的イメージと一致。
業務系（経費・精算・IT） No.8 インディゴブルー コーポレート・信頼・プロフェッショナルの青。業務系に最適。
健康系（医療・運動・健康） No.4 フレッシュグリーン 健康・自然・活力を表す明るいグリーン。医療・ウェルネス向け。
食費系（飲食・グルメ） No.3 ゴールデンイエロー 食欲・プレミアム感を演出するゴールド。食費カテゴリに直感的。
趣味系（エンタメ・趣味・娯楽） No.9 バイオレットパープル 創造性・非日常・特別感のパープル。趣味・エンタメ費に適切。
個人系（プライベート・ライフ） No.10 ローズピンク 個人・親密さ・ライフスタイルのピンク。プライベート費用に自然。
緊急系（修理・緊急出費） No.1 コーラルレッド 注意・緊急・重要性を伝える赤系。予定外の出費に視覚的インパクト。
公共系（インフラ・公共交通） No.6 ティールグリーン ブランドティールの近傍色。公共性・信頼感を表すティール系。
ブランド汎用（デフォルト・未分類） No.7 ブランドティール ロゴカラー本人。分類不明・汎用トピックのデフォルト色として使用。

/// MichiMark トピックテーマカラー 10色パレット

enum TopicThemeColor {
  // No.1 コーラルレッド (H=0°)
  coralRed(

    dark:    Color(0xFFA83030),
    tint:    Color(0xFFFADADD),

    tint:    Color(0xFFFDEBD4),
  ),
  // No.3 ゴールデンイエロー (H=60°)
  goldenYellow(
    primary: Color(0xFFC4A43A),
    dark:    Color(0xFF9A7C22),
    tint:    Color(0xFFF5ECC8),
  ),
  // No.4 フレッシュグリーン (H=90°)
  freshGreen(
    primary: Color(0xFF4DB36B),
    dark:    Color(0xFF2D8A4C),
    tint:    Color(0xFFD2F0DC),
  ),
  // No.5 エメラルドグリーン (H=155°) ← movingCost 推奨
  emeraldGreen(
    primary: Color(0xFF2E9E6B),
    dark:    Color(0xFF1A7A52),
    tint:    Color(0xFFD4F0E4),
  ),
  // No.6 ティールグリーン (H=180°)
  tealGreen(
    primary: Color(0xFF1E8A8A),
    dark:    Color(0xFF136868),
    tint:    Color(0xFFC8ECEC),
  ),
  // No.7 ブランドティール (H=200°) ← ロゴカラー
  brandTeal(
    primary: Color(0xFF2B7A9B),
    dark:    Color(0xFF1A4A5E),
    tint:    Color(0xFFD0EBF5),
  ),
  // No.8 インディゴブルー (H=225°)
  indigoBlue(
    primary: Color(0xFF3D65C4),
    dark:    Color(0xFF2245A0),
    tint:    Color(0xFFD3DCF5),
  ),
  // No.9 バイオレットパープル (H=260°)
  violetPurple(
    primary: Color(0xFF7B5CC4),

    tint:    Color(0xFFE2D8F5),
  ),

  rosePink(
    primary: Color(0xFFC4497A),
    dark:    Color(0xFF9E2A58),
    tint:    Color(0xFFF5D2E0),
  );

  const TopicThemeColor({
    required this.primary,
    required this.dark,
    required this.tint,
  });

  final Color primary;
  final Color dark;
  final Color tint;
}
