# MichiMark アイコン・スプラッシュ ブランドコンセプト

作成日: 2026-04-20

---

## コンセプトの核心

MichiMarkのアイコンは **3つのモチーフ** に集約される。

| モチーフ | 象徴するもの | 視覚表現 |
|---|---|---|
| **ミチ（道）** | 道路・旅路・作業スケジュール | 横断する道路ライン |
| **ピン** | 地点・タイミング | ロケーションピン |
| **メジャー（巻尺）** | 距離・支払い | 目盛り・ルーラー |

---

## カラー方針

### コアカラー
- ブルー系統を軸とする（現行ブランドカラー: `#1B6D8E`）
- 必要に応じてディープ方向（`#0F4A62` 系）に展開可能

### 選定理由
- ビジネス寄りのクール・落ち着いた印象
- 操作する人が慌てずに操作できる心理的安心感
- オレンジ系統も検討したが、落ち着きを優先してブルー系に決定

### スタイルルール
- **グラデーションは極力使わない**（ソリッドカラー基調）
- 光沢感・潤い感を出す加工はOK（グロスハイライト楕円など）
- シンプルな配色・多色禁止

## アイコン設計方針（B案：デザイン刷新）

### 方向性
- テキスト（"MichiMark"）をアイコンから除去（小サイズ可読性・Apple HIG準拠）
- ピン・ミチ・メジャーのコンセプトを継承しつつ洗練
- 現状より「引き算」してシンプルに統合感を高める

### 提案案（2026-04-20 v1）
- **Plan A** — ピン×道路 統合型：ピンのシルエット内部に道路を収めたミニマル案
- **Plan B** — ロードマーカー型：道路が横断し白ピンが刺さるシーン構図（直感性最高）
- **Plan C** — メジャー+ピン 抽象型：ルーラー帯の上にピンが立つ、計測・記録を前面に出した案

デザインファイル: `docs/Design/draft/michimark_icon_redesign_v1.html`

## スプラッシュ画面設計方針

### コンセプト
- **ミチが画面を横に横断するイメージ**
- その上にピンとメジャー（目盛り）が存在する構図

### レイアウト（横断構図）
```
[ 空間（ロゴエリア） ]
────────────────────── ← 道路（センターライン・サイドライン）
  📍     |    |    |   ← ピン + メジャー目盛り
──────────────────────
[ 空間 ]
```

### アニメーション構想（将来対応）
現時点は静止画。将来的なアニメーション化を想定した構造で設計する。
- 道路が左からスライドイン
- ピンが上から降りてくる
- メジャーが伸びる
