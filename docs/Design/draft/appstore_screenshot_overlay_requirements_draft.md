# デザイン要件の叩き: App Store スクリーンショット オーバーレイ

## 概要

MichiMark の App Store スクリーンショット（5枚）に、テーマカラー（Teal/Emerald）グラデーション背景 + キャッチコピー + サブテキストを重畳したオーバーレイ画像を作成する。

現状のスクリーンショット（キャプチャのみ）から、アプリの価値が一目で伝わるビジュアルへ刷新する。

## 変更項目

- [ ] 1枚目: タイムライン一覧 オーバーレイ作成（Primary Grad: #0F3D50 → #0D9488 → #059669）
- [ ] 2枚目: Mark/Link 詳細 オーバーレイ作成（Deep Grad: #0A2838 → #0F3D50 → #0D9488）
- [ ] 3枚目: 集計タブ オーバーレイ作成（Warm Grad: #0F3D50 → #0D9488 → #34D399）
- [ ] 4枚目: 支払い・精算 オーバーレイ作成（Ocean Grad: #0D9488 → #059669 → #047857）
- [ ] 5枚目: 地点追加・トピック オーバーレイ作成（Night Grad: #0F2027 → #203A43 → #2C5364）
- [ ] 各スクリーンショット: 実機キャプチャ取得（最新ビルドで再撮影）
- [ ] 出力サイズ確認: 1320 × 2868 px @3x（iPhone 6.9インチ）
- [ ] App Store Connect へのアップロード・確認

## デザイン仕様（各オーバーレイ共通）

| 要素 | 仕様 |
|---|---|
| キャンバスサイズ | 1320 × 2868 px |
| 上部エリア高さ | 全体の 28%（〜803px）— キャッチコピー + アプリ名バッジ |
| 中央エリア高さ | 全体の 55%（〜1577px）— 実機スクリーンショット + iPhone モック枠 |
| 下部エリア高さ | 全体の 17%（〜488px）— サブテキスト 2行以内 |
| キャッチコピー | 82px / 太さ 800 / 白（#FFFFFF） |
| 強調語（em） | Emerald 200（#A7F3D0）でカラー変化 |
| サブテキスト | 46px / 太さ 500 / 白 82%（rgba(255,255,255,0.82)） |
| フォント | Hiragino Sans W8（日本語） |
| テキスト占有率 | 全体の 30% 以下（Apple ガイドライン準拠） |
| コントラスト比 | WCAG AA 基準（4.5:1以上）確認済み |

## 各スクリーンショット仕様

| 枚 | 画面 | キャッチコピー | グラデーション |
|---|---|---|---|
| 1 | タイムライン一覧 | 走ったルートを、タイムラインで残す。 | Primary |
| 2 | Mark/Link 詳細 | 地点・区間・費用、まとめて記録。 | Deep |
| 3 | 集計タブ | 旅費も燃費も、自動で集計。 | Warm |
| 4 | 支払い・精算 | 割り勘精算、旅の帰りに終わる。 | Ocean |
| 5 | 地点追加・トピック | ドライブ・仕事・旅行、用途別に記録。 | Night |
