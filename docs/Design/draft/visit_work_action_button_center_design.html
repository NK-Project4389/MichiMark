実装ノート: _buildMarkCardContent() 内の分岐で topicConfig.showActionTimeButton が true の場合に中央配置レイアウトを適用する。 false の場合は従来レイアウトを維持する。Row 構成の変更は _buildCard() メソッド内で条件分岐する。
