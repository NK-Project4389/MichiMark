# /task — タスク提示スキル

このスキルが呼ばれたら、以下の手順で本日のタスクを提示せよ。

## 手順

1. `docs/Progress/README.md` を読み、最新の進捗ファイルを特定する
2. 最新の進捗ファイルを読み、「未完了の作業」と「次回やること」を把握する
3. 以下のフォーマットで本日のタスクを提示する

## 出力フォーマット

```
## 本日のタスク（YYYY-MM-DD）

### 前回からの持ち越し
- （未完了の作業を箇条書き）

### 本日の推奨タスク
1. （最優先タスク）
2. （次のタスク）
3. （余力があれば）

### 作業順の根拠
（なぜこの順序で進めるかを1〜2文で説明）
```

4. ユーザーに「この順で進めますか？変更があればお知らせください。」と確認する
